3과목 : 데이터 분석
1장: 분석 개요
1. 데이터 분석 기법의 이해
* 처리 과정:
* 데이터웨어하우스(DW)나 데이터마트(DM)으로 분석 데이터 구성
* 신규 또는 DW에 없는 데이터는 기존 운영시스템에서 직접 가져오거나 운영데이터저장소(ODS)에서 정제된 데이터를 가져와 DW의 데이터와 결합해 이용
* 시각화 기법:
* 가장 낮은 분석이나 잘 활용하면 복잡한 분석보다 더 효율적. 대용량 데이터 다룰 때와 탐색적 분석을 할 때 필수
* 공간분석(GIS):
* 공간적 차원과 관련된 속성들을 시각화하는 분석.
* 지도 위에 관련 속성들 생성 후 크기, 모양, 선 등으로 구분하여 인사이트를 얻음
* 탐색적 자료분석(EDA: Exploratory data analysis)
* 다양한 차원과 값을 조합해 가며 특이점이나 의미있는 사실을 도출하고 분석의 최종목적을 달성해가는 과정
* EDA의 4가지 주제: 저항성의 강조 / 잔차 계산 / 자료변수의 재표현 / 그래프를 통한 현시성 
* 통계 분석
* 어떤 현상을 종합적으로 한 눈에 알아보기 쉽게 일정한 체계에 따라 숫자, 표, 그림의 형태로 나타내는 것
* 기술통계: 모집단으로부터 표본을 추출하고 표본의 가지고 있는 정보를 쉽게 파악할 수 있도록 데이터를 정리하거나 요약하기 위해 하나의 숫자, 또는 그래프의 형태로 표현
* 추측/추론 통계: 모집단으로부터 추출된 표본통계량으로부터 모집단의 특성인 모수에 관해 통계적으로 추론
* 데이터 마이닝
* 대용량의 자료로부터 정보를 요약하고 미래에 대한 예측을 목표로 자료에 존재하는 관계, 패턴, 규칙 등을 탐색하고 이를 모형화하여 이전에 알지 못한 유용한 지식을 추출
* 방법론: 기계학습(ANN, DecisionTree, clustering, SVM), 패턴인식(연관규칙, 장바구니분석 등)
* 데이터 마이닝 모델링 성능평가 기준
* 정확도
* 정밀도
* Detect rate
* Lift

* Ex. 세분화, 고객행동 예측, 캠페인 반응 예측 등

1. 기초 분석 및 데이터 관리
* 데이터 탐색적 자료 분석 (EDA)
* 분석에 앞서 전체적으로 특징 파악
* 다양한 각도로 접근
* summary()함수로 기초 통계량 확인
* 결측값 처리 방법
* 단순 대치법
: completes analysis: 결측값 레코드 자체를 삭제
: 평균 대치법 (Mean imputation): 데이터의 평균으로 대치
: 단순 확률 대치법: 평균대치법에서 추정량 표준 오차의 과소추정문제를 보완코자 고안됨. Hot-deck, nearest neighbor 방법
* 다중 대치법
: 단순대치법을 한 번에 하지 않고 m번의 대치를 통해 m개의 가상적 완전 자료를 만드는 방법
: 대치 -> 분석 -> 결합
* R에서의 결측값 처리 함수
* complete.cases(): is.na()와 반대, 결측값 있을 때 FALSE임
* DMwR.centralImputation(): NA값에 가운데 값으로 대치. Numeric data는 중위수, 요인은 최빈값
* DMwR.knnImputation(): NA값을 k-nearest-neighbor로 대치 
* Amelia.amelia()
* 이상값 인식방법
* ESD(Extreme Studentized Deviation)
: 3 sigma(평균으로부터 +/-3표준편차)
: 기하평균 -2.5*sigma < data < 기하평균 + 2.5*sigma
: 사분위수 이용하여 제거 (boxplot IQR 외부 값): Q1-1.5(Q3-Q1) < data < Q3 - 1.5(Q3-Q1)
* 극단값 절단 방법
* 기하평균 이용 제거 : geo_mean (n_th root(a1*a2*...an)
* 하단, 상단 각각 5%씩 절단
* 극단값 조정 방법
* 상한값과 하항값을 벗어나는 값들을 각각의 상한값, 하한값으로 대치
4장: 통계분석
1. 통계분석의 이해
* 표본 조사
* 모집단, 원소(모집단 구성 개체), 표본, 모수(표본관측으로 구하고자 하는 모집단에 대한 정보)
* 표본 추출 방법
* 단순랜덤추출법: 각 샘플에 번호 부여, 임의의 nro 추출
* 계통추출법: k개씩 띄어서 표본 선택. 매 k번째 항목 추출
* 집락추출법: 군집별로 단순랜덤 추출
* 층화추출법: 유사한 원소끼리 몇 개의 층으로 나누어 각 층에서 랜덤 추출
* 측정 방법
* 명목척도: 측정 대상이 어느 집단에 속하는지 (성별, 출생지, 참석 비참석 등)
* 순서척도: 측정대상의 서열관계 (만족도, 선호도, 학년 등)
* 구간척도: 측정대상이 갖고 있는 속성의 양을 측정, 구간과 구간사이 간격에 의미를 둠 (온도 0-5도, 5-10도 에 속하는 국가들(??) 등)
* 비율척도: 간격에 대한 비율의 의미, 절대적 기준 0이 존재, 사칙연산 가능
* 통계분석
* 특정 집단이나 불확실한 현상을 대상으로 자료를 수집해 대상 집단에 대한 정보를 구하고, 적절한 통계분석을 이용해 의사결정
* 기술통계
* 통계적 추론(추측 통계)
* 수집된 자료를 이용해 대상 집단에 대한 의사결정
*  모수추정 / 가설검정  / 예측
* 이산형 확률변수: 0이 아닌 확률값을 갖는 확률 변수를 셀 수 있는 경우
* 베르누이 확률분포: 결과가 2개만 (1시간 이상 기다리는 고객이 남자냐 여자냐)
* 이항분포: 베르누이를 n번 반복시 k번 성공할 확률
* 기하분포: 성공확률이 p인 베르누이 시행에서 첫 성공까지 x번 실패할 확률
* 다항분포: 이항분포의 확장, 세 가지 이상의 결과를 가지는 반복 시행에서 발생 확률 분포
* 포아송분포: 시간과 공간 내에서 발생하는 사건의 발생횟수
* 연속형 확률변수: 가능한 값이 실수의 어느 특정구간 전체에 해당하는 확률변수
* 균열분포: 모든 확률변수 X가 균일한 확률을 가지는 확률분포
